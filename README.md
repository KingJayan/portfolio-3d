# 3d portfolio hub

### description
a single-file, immersive 3d portfolio experience. minimal, dark-industrial aesthetic with a focus on spatial interaction and fluid camera movement.

### constraints
- **single-file**: no external assets (images, fonts, or libraries).
- **vanilla-only**: zero dependencies. pure html, css-3d, and js.
- **performance**: manual camera interpolation and distance-based proximity triggers.

### stack
- **html5**: semantic tags and 3d coordinate storage.
- **css3**: dark-industrial neumorphism, 3d transforms, and keyframe animations.
- **javascript**: 6-dof camera control, vector-based scrolling, and click-to-focus logic.

### what its for
curating engineering projects, experimental prototypes, and architectural artifacts in a persistent spatial environment.

### needed
- modern browser with hardware acceleration.
- mouse/trackpad (recommended for 3d navigation).
- `index.html` (the entire project).
